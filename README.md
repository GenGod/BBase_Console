BBase

Программа пригодится завхозу овощебазы. Полиморфизм на данной стадии реализован в в виде функции, выдающей разные сообщения на этапе добавления нового элемента в базу, в зависимости от типа добавляемого элемента.

https://github.com/GeneralGod/BBase_Console/issues/1 - коментарии и обсуждение программы
